What is Morse Code?

Morse Code is a language that represents words and phrases as a series of dots dashes and spaces.

Morse Code was invented in 1837, at a time where there was no possible way to instantly send written text to far off cities. The code's creators were Samuel F. B. Morse, Joseph Henery, and Alfred Vail. These men took advantage of the telegraph system by creating a language made up of long and short electromagnetic signals. These signals could then be sent through wires so that messages could travel quickly even via long distances.


How to Read Morse Code ...

The code consists of five different parts:

  • Dots (or "dits") which are short signals
  • Dashes (or "dahs") which are long signals
  • Short Pauses which go between letters
  • Meduim Pauses which go between words
  • Long Pauses which go between sentences
